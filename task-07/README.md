# wildberries task L2

Реализация игры «Угадай число». 

[Деплой](https://dulcet-genie-e29fc8.netlify.app/)

## Функционал
Пользователь может:
* Указывать диапазон загадываемого числа (целые числа).
* Перезапускать игру.
* Угадывать число...

## Стек технологий
В проекте используется только голый js. css, html для верстки.
